# TEST_TASK

Тестовое задание: мини-модуль "Список задач с просмотром деталей"

## ✨ Функционал

### Основной
| Функция | Описание |
|---------|----------|
| 📥 **Загрузка задач** | Получение данных с JSONPlaceholder API (первые 10 задач) |
| 📋 **Список задач** | Отображение задач в виде карточек с названием и статусом |
| 🔘 **Кнопка статуса** | Изменение статуса задачи без открытия панели (круглая кнопка слева) |
| 🔓 **Кнопка "Открыть"** | Открытие боковой панели с деталями задачи |
| 📌 **Боковая панель** | Детальный просмотр задачи с названием, описанием и статусом |
| 🔄 **Переключение статуса** | Возможность изменить статус прямо в панели |
| ❌ **Кнопка "Закрыть"** | Закрытие панели и возврат к списку |

### Дополнительный
| Функция | Описание |
|---------|----------|
| 🔍 **Фильтрация задач** | Фильтр "Все / Активные / Выполненные" |
| 📊 **Статистика** | Счетчики общего количества, выполненных и активных задач |
| 🔒 **Блокировка скролла** | Запрет прокрутки фона при открытой панели |

## 🛠 Технологии

| Технология | Версия | Назначение |
|------------|--------|------------|
| **React** | 18.2.0 | Библиотека для построения интерфейса |
| **TypeScript** | 5.0.0 | Типизация кода, предотвращение ошибок |
| **Axios** | 1.6.0 | HTTP-клиент для запросов к API |
| **CSS Modules** | - | Изоляция стилей компонентов |
| **JSONPlaceholder** | - | Тестовое REST API |
